ominaisuudet

- mahdollisimman vähän epäsäännöllisyyksiä

- savolainen viäntäminen

- verbin perusmuoto päättyy -au tai -äy
  soutaa => soutau
  syödä  => syödäy
  juoda  => juodau
  olla   => ollau 

- verbin hän-muoto -hu/y pääte
  hän opastaa => hän opastaahu
  hän syö = hän syöhy

- adjektiivin monikko -het pääte  
  valmiit => valmihet
  kauniit => kaunihet

äiti = muamo, emo
hän = häi
itsehän = itsehäi

- käskyn monikko -ttakkua pääte (perusmuoto - pääte + -ttakkua)
  laskekaa => laskettakkua
  syökää   => syöttäkkyä
  kuunnelkaa => kuunneltakkua

- verbin imperfekti -ai pääte
  lyödä -> löi => löittäi
  osti => ostai
  luki => lukai
  luin => lukain
  luit => lukait
  luimme => lukaimme
  luitte => lukaitte
  lukivat => lukaivat

- konditionaalin -aisiin pääte => ahin 
  hampaisiin => hampahin
  voitaisiin => voetahin
  syötäisiin => syötähin
